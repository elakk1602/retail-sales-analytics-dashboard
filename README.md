# Real-Time Retail Sales Analytics Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze sales performance, customer trends, profit analysis, and regional insights using retail business data.

## Features
- KPI Tracking
- Sales Forecasting
- Regional Analysis
- Interactive Filters
- Customer Insights
- Profit Analysis

## Tools Used
- Power BI
- Excel
- DAX
- Power Query
- Data Visualization

## Dashboard Insights
- Analyzed regional sales performance
- Identified top-performing products
- Tracked monthly revenue trends
- Monitored profit and sales KPIs
